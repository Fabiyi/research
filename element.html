50 HTML ELEMENTS AND THEIR FUNCTIONS

  • Header Tag

    • It contains meta-information about their document such as the title, character set, links to style sheets, and scripts.

  • Paragraph

    • it is used to define block of text.

  • Break line

    • It is used to start a new line within a paragraph.

  • Horizontal rule

    • It is used to create a thematic break or a division between sections (self-closing).

  • Strong

    • It is used to bold important text usually bold by default.

  • Emphasizes text

    • It emphasize text usually italicized by default.

  • Anchor

    • An anchor is used to create hyperlinks.

  • Unordered list

    • It typically rendered with bullet points.

  • Ordered list

    • It typically rendered with numbers or letters.

  • List items

    • It is used within ordered list or unordered list

  • Table

    • It is used to define a table.

  • Table row

    • It is used to define the table row.

  • Table data

    • It is used to define the table data cell used within an element.

  • Table header cell

    • It is used within an element typically bold and centered.

  • Form

    • It defines a form for user input. The action attribute specifies where to send the form.

  • Input

    • It define the input field for user data common types include text, password, submit, radio, checkbox etc.

  • Select

    • It define the drop down list of the data.

  • Option

    • It It define the individual option within an element.

  • Image

    • It is used to embed an image in the page.

  • Audio

    • It is used to embed audio content

  • Video

    • It is used to embed video content

  • DIV

    • It is block level container for grouping elements.

  • Span

    • It is an inline container for group element but it does not cause line break.

  • Center Tag

    • It is used to center text or data

  • Font Color

    • It is used to change colour of the text

  • Font Size

    • It is used to change the size of the text.

  • Font Face

    • It is used to change the appearance of the text.

  • Big Tag

    • It is used to increase the size of the text or data

  • Small Tag

    • It is used to reduce the size of the text or data

  • Quoting Passage of Text

    • It is used for encloses text.

  • Field set Tag

    • It is used for enclose part of a webpage

  • Legend Tags

    • It is used for enclose text of a webpage.

  • iframe

    • It allows you to embed another webpage or URL inside your web page.

  • Superscript

    • It is assign text or data

  • Body

    • It contains the content of the webpage.

  • Style

    • It is used to add JavaScript to the webpage.

    • It is used to declare the document type and version of HTML.

  • HTML

    • It is the root element that contains all other elements on the page.

  • Text area

    • It is used to define a multiline text input.

  • Button

    • It create a clickable button on the webpage.

  • Source

    • It is used to define multiple media resources for video or audio.

  • Span

    • It is a generic inline container for text, often for styling.

  • Emphasizes

    • It is used to emphasize text usually italic.

  • Navigation

    • It contains navigation links.

  • Footer

    • It define a footer for a document or section.

  • Preformatted

    • It allows typed text to appear on the webpage as typed.

  • Script

    • It embeds or refers to a script usually in JavaScript.

  • Label

    • It defines a label for an input element.

  • Aside

    • It define content aside from main content like a sidebar.

  • Meta

    • It provides metadata about the HTML document e.g character set, description, keyword.
